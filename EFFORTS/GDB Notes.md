# GDB Commands

- disable \<br no.>
- dprintf \<line no.>, _printing condition_, variable
  ex: dprintf 8, "var=%d\n" , var
- watch _variable_
  stops when the variable value changes. 
  ex: watch $sp
- help
  gather info about a command
